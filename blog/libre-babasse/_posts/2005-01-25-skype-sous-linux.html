--- title: Skype sous Linux ---

Si vous avez une amie qui vous est très chère, mais qui habite loin (genre en Irlande par exemple), et que vous voudriez bien pouvoir l'appeler pour pas cher, vous avez l'option France Telecom. Pour encore moins cher (quoiqu'il faut quand même un abonnement internet aux deux bouts), vous avez aussi l'option Skype.

\r\n\r\n

J'ai un peu galéré pour le faire marcher alors voilà ce que j'ai fais, si cela intéresse quelqu'un. Tout d'abord, pour installer le logiciel en lui-même, on peut télécharger le binaire chez Skype, et cela marche plutôt bien. Mais si vous aimez vos packages Debian, vous pouvez utiliser la ligne suivante (à ajouter à votre fichier /etc/apt/sources.list) : deb http://debian.neo.pl/wfmh unstable main contrib non-free. Ensuite, vous pourrez tout simplement installer automatiquement skype avec sudo apt-get update et sudo apt-get install skype. Ca, c'est fait.

\r\n\r\n

Il faut ensuite régler la traversée du pare-feu. Le principe est de trouver quel port est utilisé par Skype et de laisser entrer toutes les paquets UDP et TCP sur ce port. Je vous propose de faire ça avec l'excellent Firehol.

\r\n\r\n

Ce que l'on va faire, c'est définir un service skype et ensuite autoriser ce service en entrée. Cette configuration se fait dans /etc/firehol/firehol.conf. Pour spécifier le service, il faut ajouter les deux lignes suivantes :

\r\n\r\n

\r\n\r\nserver_skype_ports="udp/XXXXX tcp/XXXXX"
\r\nclient_skype_ports="any"\r\n\r\n

\r\n\r\n

où XXXXX est le numéro de port que vous pourrez trouver dans l'onglet Advanced... du panneau Options de Skype (menu Fichier), sous la désignation Utiliser le port XXXXX pour les connexions entrantes. Toujours dans le fichier /etc/firehol/firehol.conf, vous pouvez ensuite utiliser le service skype comme vous utilisez n'importe quel autre service défini par défaut dans Firehol.

\r\n\r\n

Et pour vérifier que tout marche maintenant bien, vous pouvez utiliser le compte Skype echo123, qui répondra tout le temps, vous laisse parler, et vous rejoue ce que vous avez dit. Pour finir, si vous avez des soucis de son, vous pouvez regarder la documentation chez Skype.
